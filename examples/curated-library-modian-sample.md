# 专题精选索引 · 样例（虚构）

本文件是 `$CURATED_LIBRARY_ROOT/example-topic/_index.md` 的**虚构样例**，供 GitHub 读者参考。  
**可编辑的实盘路径在本地精选库**，不在本仓库内嵌 PDF。

## 设计要点

1. **不搬全家当**：原件仍放 Obsidian / 本地资料目录，索引表记录权威路径
2. **Agent 先读索引**：再 `gety search -c "Folder: 精选库"`，最后全盘
3. **专题名自定**：`example-topic` 可换成你的任意主题目录名

## 试卷结构（示例）

- 选择 30 + 简答 40 + 大题 80 = 150

## 章节权重（复习优先级 · 示例）

1. 高频章节 A（15–30 分）
2. 高频章节 B（可出大题）
3. 简答常考章节
4. 其余章节按 `_index.md` 表排序

## 本地实盘

完整路径表见本地：

```

```
